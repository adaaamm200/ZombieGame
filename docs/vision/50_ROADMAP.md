# Roadmap (Milestones)

High-level milestones. Detailed phases live in [`../MASTER_PLAN.md`](../MASTER_PLAN.md);
live status in [`../STATUS.md`](../STATUS.md).

- **M0 — Memory + agent system** (this milestone): persistent project memory,
  agent role cards, asset pipeline docs, visual QA rules.
- **M1 — UI finalization**: main menu, campaign board, modals, HUD, icons — premium,
  consistent, reference-matched.
- **M2 — Day board system**: day-based campaign structure, mission states, progression.
- **M3 — Day 1 gameplay backgrounds**: premium in-match stage backgrounds.
- **M4 — Playable character v1**: first character sprite in game.
- **M5 — Zombie v1**: first improved zombie type in game.
- **M6 — More characters / zombies**: expand roster and enemy variety.
- **M7 — Weapon / upgrade polish**: weapon cards, upgrades, progression feel.
- **M8 — Combat effects**: muzzle, tracer, hit, explosion, screen shake, juice.

> Do one milestone at a time; the owner tests after each. A failed milestone gets a
> fix round, not a new feature.
