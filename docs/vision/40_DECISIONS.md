# Decisions (Recorded)

Settled decisions. Do not re-litigate without explicit owner approval.

## Campaign structure
- **Day-based campaign.**
- **1 Day = 5 missions.**
- **Day finale = boss** (mission 5).
- 100 days possible long-term, but **not now** as weak, repetitive content.
- A Day is only cleared when all 5 missions are done; the next Day unlocks after the finale.
- Free Mode / Scavenge is separate and does **not** unlock campaign.

## Postponed
- Keycards / blueprints — **postponed**.

## Pickups (MVP)
- Keep simple: **coin, ammo, medkit, grenade.**

## Weapon line (progression)
- pistol → SMG → assault rifle → shotgun → sniper → LMG/minigun → laser →
  rocket launcher → flamethrower.

## Visual / UI (see also 20_NON_NEGOTIABLE_RULES)
- No generic roadmap board; illustrated premium board + hotspots.
- Reference images take priority; real board/background images are used as assets.
- Old green prototype UI dropped; green only for ready/success/health.
- Icons: safe padded box, centered, uniform; no fragile sheet crop.

## Process ordering (avoid mixing too many threads)
Memory/agent system → visual audit → crop fix → UI finalization → asset pipeline →
sprite implementation (in that order).
