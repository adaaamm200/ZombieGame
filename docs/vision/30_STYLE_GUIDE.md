# Style Guide

## Mood
- Dark cinematic survival horror.
- Gritty urban apocalypse.
- Military / survivor tone.
- Premium mobile-game UI (dark metal, smoky glass, beveled, high-contrast).

## Color system (UI)
- **Gold** = primary action / reward / shop / value.
- **Red** = boss / danger / kill / infected.
- **Green** = ready / success / health.
- **Purple** = loot / scavenge / special mode.
- **Blue / cyan** = lab / utility / info.
- **Dark metal** = base UI housing.
- **No random green dominance** — green is a state accent only, not the base skin.

## UI treatment
- Beveled, high-contrast, mobile-game-style buttons.
- Bold, readable typography; larger rather than thin.
- Thick, bright, readable borders; subtle glow; depth (top light + bottom shadow).
- Icons live in a **safe padded box**, centered, uniform, object-fit: contain.

## Board / markers
- Illustrated / pseudo-isometric premium board with hotspot overlay.
- Hexagon mission markers, colored per state, with readable name labels.
- Boss marker = large, red, threatening. Scavenge = purple loot. Locked = dark but readable.

## Characters / zombies (later)
- Original designs, transparent background, consistent baseline.
- Pipeline: concept sheet → turnaround → sprite sheet → implementation brief.
