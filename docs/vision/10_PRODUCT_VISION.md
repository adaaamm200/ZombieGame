# Product Vision

**ZombieGame / Zombi Krónika / Zombie Chronicles** — a dark, gritty, premium
zombie arcade shooter.

## Core
- HTML5 Canvas + vanilla JS + PWA, zero-dependency, no build step.
- Zombie Diary **inspiration**, but with fully original name, art, UI and assets.
- No copyrighted assets, UI, characters, audio or names.

## Pillars
- Strong visual quality and good **game feel**.
- **Day-based campaign** (1 Day = 5 missions, 5th = boss finale).
- **Scavenge / Free Mode** (endless farm, does not unlock campaign).
- **Weapon progression** across a full weapon line.
- Characters, zombies and sprite polish come **later** (concept sheet → turnaround →
  sprite sheet → implementation brief).
- Mobile-friendly, enjoyable in landscape.

## Anti-goals (do NOT)
- No generic roadmap / 1–40 node chain campaign map.
- No old green prototype UI dominance (green only for ready/success/health).
- No tiny centered canvas / shrunk viewport.
- No aggressive live-service UI: no Battle Pass, gems, clans, ranking, starter packs.
- No feature bloat; ship coherent milestones.
