# Weapon asset sheet és külön fegyverképek

Referencia: `visual_references/weapon_asset_sheet_reference.png`

## Cél

A fegyverek ne maradjanak placeholder pálcika/pixel blokkok. Külön fegyverképek kellenek, azonos stílusban.

## Szükséges assetek

Minden fegyverhez:

- teljes fegyver render PNG
- kártya thumbnail PNG
- opcionális silhouette / low-detail fallback
- icon badge / ammo type icon
